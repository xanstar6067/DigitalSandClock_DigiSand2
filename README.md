# Sand clock for STM32G031F6P6

Проект цифровых песочных часов перенесён с STM32F411CEU6 на
STM32G031F6P6 (TSSOP20) и предназначен для STM32CubeIDE.

Две последовательно соединённые матрицы MAX7219 образуют дисплей 16×8.
MPU6050/MPU6500 определяет ориентацию устройства, после чего прошивка
моделирует пересыпание 55 частиц между половинами дисплея.
При холодном старте движение блокируется до первого успешного измерения MPU;
после готовности датчика песок инициализируется автоматически.

## Подключение

| Сигнал | STM32G031F6P6 | Режим |
| --- | --- | --- |
| MAX7219 DIN | PA1 | GPIO output |
| MAX7219 CS | PA2 | GPIO output |
| MAX7219 CLK | PA3 | GPIO output |
| ULED | PA4 | GPIO output, active-low |
| EXT_BTN1 | PA5 | GPIO input, pull-up, кнопка на GND |
| EXT_BTN2 | PA6 | GPIO input, pull-up, кнопка на GND |
| UKEY | PA14/PA15, общий вывод корпуса | GPIO input, pull-down, кнопка на 3,3 В |
| MPU SCL | PB6 | I2C1 SCL |
| MPU SDA | PB7 | I2C1 SDA |
| BUZZER PWM N | PA7 | TIM1_CH1N, зарезервирован |
| BUZZER PWM | PA8 | TIM1_CH1, зарезервирован |

SWD использует PA13 (DIO) и PA14 (CLK). В корпусе TSSOP20 PA15 и PA14
используют общий вывод корпуса, к которому на плате также подключена кнопка
KEY с внешним pull-down 10 кОм. После запуска прошивки этот вывод становится
входом UKEY. Для повторного подключения отладчика при необходимости
используйте режим Connect under reset.

## Управление

- EXT_BTN1: короткое нажатие добавляет 1 секунду, удержание — по 10 секунд.
- EXT_BTN2: короткое нажатие убавляет 1 секунду, удержание — по 10 секунд.
- EXT_BTN1 + EXT_BTN2: сброс песка.
- UKEY: короткое нажатие сбрасывает песок, удержание меняет яркость.

## Flash и сборка

Последняя страница Flash размером 2 КБ
(`0x08007800..0x08007FFF`) зарезервирована для времени и яркости.
Поэтому в `STM32G031F6PX_FLASH.ld` область приложения ограничена 30 КБ.
После повторной генерации кода в CubeMX/CubeIDE проверьте, что это
ограничение линкерного скрипта сохранилось.

Проверенные размеры:

- Debug (`-O0`): 29 612 байт Flash, 1 944 байта BSS;
- Release (`-Os`): 18 508 байт Flash, 1 944 байта BSS.
